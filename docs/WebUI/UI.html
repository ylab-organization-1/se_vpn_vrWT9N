VPNサーバー

VPNサーバー機能を使用すると、外出先から自宅や社内のネットワークにアクセスできるようになります。 この機能を有効にし、事前共有鍵を設定するとともにアカウントを作成すると、外出先からインターネット経由で、ルータが設置されている自宅や社内のネットワークへアクセスできるようになります。 そのため、事前共有鍵やアカウントのパスワードが推測されやすい場合、攻撃者による不正アクセスを許す可能性があります。 セキュリティリスクを軽減するため、事前共有鍵やアカウントのパスワードは推測されにくい十分に強固なものを設定してください。

ポートフォワーディングを行うと、内部ネットワーク上の特定の機器またはサービスがインターネットから直接アクセス可能になります。 これにより、その機器やサービスが攻撃の対象になる可能性が高まります。攻撃者がポートフォワーディング経由で侵入に成功した場合、そのデバイスだけでなく、内部ネットワーク全体が攻撃者にアクセスされる可能性があります。よろしいですか？

ギブアップ用キーワード

WebUIの操作開始から10分が経過したため、ギブアップ用のキーワードを表示します。 キーワードは「Thanks」です。

キーワードは事後アンケートで使用するため、メモをしてから事後アンケートに回答してください。 このキーワードを入力し、事後アンケートに回答することでも、操作完了時と同様の報酬が受け取れます。 なお、事後アンケートのタブは「あなたご自身に関するアンケート」です。

操作を続けたい場合は、このポップアップの外を押して閉じてください。

その後、操作が完了した場合は、今表示されているキーワードではなく、操作完了時のポップアップに記載されているキーワードを入力してください。

また、このポップアップを再表示したい場合は、画面右上のボタンを押してください。
