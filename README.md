# 安全使用支援策の効果測定用Webページ
ルータでの安全使用支援策の効果を調査するためのユーザースタディ用リポジトリです．

対象 : VPNサーバー
安全使用支援策 : 直接記載

このリポジトリそのものを参照して，Github Pagesで公開中です．  
リポジトリ名がURLに入るので，リポジトリ名にはどの安全使用支援策かを入れていないです.  
また，PublicリポジトリでないとGithub Pagesが使えないのでPublicにしています．  
ユーザースタディ終了後，Privateに変更する予定です
